## Tech Stack

- **Frontend:**
  - Next.js 14
  - React 18
  - TailwindCSS
  - shadcn/ui components
  - React Quill for rich text editing

- **Backend:**
  - Supabase (PostgreSQL + Real-time subscriptions)

- **Additional Tools:**
  - TypeScript
  - next-themes for dark mode
  - react-hot-toast for notifications
  - Lodash for debouncing
